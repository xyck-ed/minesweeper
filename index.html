Click Flag To Mark A Mine
MINESWEEPER
DIFFICULT
INSTRUCTIONSYour mission, should you choose to accept it, is to map an active minefield as quickly as possible, without exploding any of the mines. Begin by clicking any grid. If the grid does not contain a bomb, it will either display how many bombs currently surround it or display a blank if no bombs are detected nearby. If you suspect a grid contains a mine, click the flag BEFORE clicking the grid in question. This marks the grid as dangerous. The game is over when every grid has either been cleared or correctly marked with a flag. GOOD LUCK!
